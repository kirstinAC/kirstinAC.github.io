HTML5 video player

5.0/5 rating (1 votes)

For me Gantry 4 was released just in time. Two days earlier, I got a new job in order to make a website.
One of the items on this page are the training videos. Gantry 4 is HTML 5 compatible so I can use new standard for media. Unfortunately default players are different in each web browser and do not look very nice. In web is plenty js extensions for media player. I have chosen Video.js due to it plays also Youtube and Vimeo movies.

Before we start few words about video standrad in HTML5. There is no until now one common video format for all browsers. See table below:

BrowserOgg Theora (ogv)MP4WebM
Firefox 4.0+

Safari 3+

Chrome 6 +

Chrome 5 -

Opera 10.6 +

Opera 10.5

IE 9 +

I'm going to use mp4 and ogg Theora formats. If you need video converter there is one nice and small Miro Video Converter

Of course we start with the fields

     <fields name="html5player" type="chain" label="HTML5_PLAYER">
         <field name="enabled" type="toggle" default="0" label="ON/OF"/>
         <field name="internal" type="toggle" default="0" label="INTERNAL"/>
         <field name="youtube" type="toggle" default="0" label="YOUTUBE"/>
         <field name="vimeo" type="toggle" default="0" label="VIMEO"/>
    </fields>

What is "internal"? Videojs can be used with links to js and css files on ofiicial site. If you switch this toggle to internal js and css from your server will be linked in header.

Now our feature

<?php
/**
 * @package     gantry
 * @subpackage  features
 * @version        4
 * @author      Marcin Paczkowski
 * @license     do whatever you want :)))
 */

defined('JPATH_BASE') or die();

gantry_import('core.gantryfeature');

class GantryFeatureHtml5player extends GantryFeature {
    var $_feature_name = 'html5player';

    function init() {
    global $gantry;

$Folderjs = JURI::root(true)."/templates/".strtolower($gantry->get('template_full_name'))."/js/html5player";

		if ($gantry->get('html5player-enabled')) {


			if($gantry->get('html5player-internal')==1) {

			$gantry->addLess('video-js.less');
			$gantry->addScript($Folderjs ."/video.js");	}

			else {

			$gantry->addStyle('//vjs.zencdn.net/c/video-js.css');
				$gantry->addScript('//vjs.zencdn.net/c/video.js');}

			if($gantry->get('html5player-youtube')==1)
			{$gantry->addScript($Folderjs ."/html5.js");
			$gantry->addScript($Folderjs ."/youtube.js");}

			if($gantry->get('html5player-vimeo')==1)
			{$gantry->addScript($Folderjs ."/html5.js");
			$gantry->addScript($Folderjs ."/vimeo.js");}

			}
		}

	}

Very easy - adds relevant links to css and js files depending on which toggle is off/on

Video code

<video id="my_video_1" class="video-js vjs-default-skin video-big"
 controls  preload="auto"   poster="/images/greenday.jpg"
 data-setup="{}" >
<source src="/images/greenday - Green Day - 21 Guns.mp4" type='video/mp4'>
<source src="/images/greenday - Green Day - 21 Guns.theora.ogv" type='video/ogv'>
</video>
Result

Green Day - "21 Guns"


Parameters:
"controls" - shows controls like play, pause etc
"poster" - image shown before video is playing
"preload" - preloads video when page is open. You can use "auto", "metadata"-loads only metadata and "none"
"autoplay" - will play video just after page is loaded
"width" & "height" - no need to explain :))

Youtube and Vimeo

Code for youtube

<video id="example_video_2" class="video-js vjs-default-skin video-small" controls
 data-setup='{"techOrder":["html5", "youtube"]}'>
<source src="http://www.youtube.com/watch?v=kL0ASfyPIuU" type='video/youtube'>
</video>

Code for vimeo

<video id="example_video_3" class="video-js vjs-default-skin video-small" controls

</video>

Billy Talent - "Dead Viking March"

Styles

You may say that it doesn't look very good. That's true - by default there is no big difference to default browser player. If you open video-js.less file you will find out that skin is built by regular css. Controlers are regular pngs, progress bars are gradients. It is quite easy to customize style to your own preferences. In fact this is second reason why I'm using Videojs.


There is more options which possible to use with Videojs. More information you can find on the official page Videojs

Latest update

To video-js.less I have added 2 resposinve classes - video-big (movie 1) and video-small (movies 2 adn 3). If you will use those clasess there is no need to use parameters "width" and "height". Of course if you want you can add more classes.
